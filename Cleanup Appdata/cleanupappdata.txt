Docker Not Running=Servizio Docker non in esecuzione
Because the docker service is not currently running, this module will prompt you to delete the appdata for any/all docker applications which are installed=Poiché il servizio Docker non è attualmente in esecuzione, questo modulo ti chiederà di eliminare i dati delle app per tutti i Contenitori Docker installati.

Deleted=Eliminazione completata!
The following appdata paths have been determined based upon your previously installed applications to be no longer in use Select which ones you would like to be deleted=I seguenti percorsi '/appdata' sono stati identificati (in base alle applicazioni precedentemente installate) come non più in uso. Seleziona quali desideri eliminare.
Note that this module looks at all of your previously installed docker applications to determine which appdata folders are abandoned If you have deleted the previously installed list either through Community Applications or manually, then it will not display those result In other words, if there is not a record for the folder within the previously installed apps, then you will not be prompted to delete the folder=Nota che questo modulo esamina tutti i Contenitori Docker precedentemente installati per determinare quali cartelle '/appdata' non sono più in uso. Se hai eliminato l’elenco delle applicazioni precedentemente installate (sia tramite Community Applications, sia manualmente), allora non verranno mostrate nei risultati. In altre parole, se non esiste un registro della cartella nelle applicazioni precedentemente installate, non ti verrà chiesto di eliminare tale cartella.
In case of a severely misconfigured app previously installed, the entire contents of a share MAY be offered up to delete It is your responsibility to confirm all of the folders being offered up to delete If you inadvertently delete a share which you actually do want to keep due to your own mistakes in configuring an application, the deletion is not reversable and the author of the plugin will take no responsiblility=In caso di un’app precedentemente installata ma configurata in modo errato, l’intero contenuto di una Condivisione **POTREBBE** essere consigliato per l’eliminazione. È tua responsabilità controllare tutte le cartelle che ti vengono suggerite. Se elimini involontariamente una Condivisione che in realtà desideri mantenere, tieni presente che l’operazione è **IRREVERSIBILE** e che l’autore del plugin non si assume alcuna responsabilità in merito.
