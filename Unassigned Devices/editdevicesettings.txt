:ud_edit_script_attach_plug:
> Clicca o premi su **Collega** per collegare un dispositivo che è stato rimosso. Il dispositivo apparirà come appena installato e verrà montato in automatico se l’opzione 'Monta Automaticamente' è attiva. È necessario un po’ di tempo prima che il dispositivo appaia in Unassigned Devices.
:end

:ud_edit_script_abort_plug:
> Clicca o premi su **Annulla** per annullare lo Script del Dispositivo.
:end

:ud_disk_name_plug:
> Specifica un nome per questo dispositivo, verrà utilizzato per fare riferimento a questa unità. Lasciando il campo vuoto sarà assegnato il nome predefinito '**devX**'.
>
> **Nome:** massimo 8 caratteri; gli spazi vengono convertiti in '_'; alcuni caratteri speciali sono consentiti; il nome è 'case-sensitive'.
>
> **NOTA:** Il nome non può iniziare con '**dev**' in quanto è riservato per la nomenclatura predefinita '**devX**'.
:end

:ud_edit_script_choose_name_plug:
> Questo è il File Script che viene eseguito quando il dispositivo viene montato. Scegli il nome del tuo File Script dal menu di selezione file, oppure inserisci lo script desiderato.
:end

:ud_edit_user_script_choose_name_plug:
> Se hai installato il plugin '**User Scripts**', Unassigned Devices
può mostrare lo stato di esecuzione dello script che userà questo dispositivo. Questo campo è abilitato solo quando esiste un File Script del Dispositivo.
:end

:ud_edit_script_run_background_plug:
> Imposta su **Sì** per consentire allo Script del Dispositivo di essere eseguito in background. Questa operazione evita che Unassigned Devices si blocchi quando lo script viene eseguito dall’Interfaccia Utente di Unraid.
:end
