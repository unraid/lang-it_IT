Hide Deprecated Applications=Nascondi app obsolete
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=Questa impostazione nasconde le applicazioni obsolete dall’elenco delle app disponibili. Tali app verranno comunque visualizzate nelle sezioni 'App installate' e 'App precedenti'.
Hide Incompatible Applications=Nascondi le app non compatibili
This setting will hide applications that are listed as being incompatible with your version of Unraid=Questa impostazione nasconderà le applicazioni elencate come incompatibili con la tua versione di Unraid.
Allow install of second instance=Consenti l’installazione di una seconda istanza
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=Questa opzione abilita un altro pulsante sulle schede e sui popup che ti permetterà di reinstallare un’applicazione già installata usando i valori predefiniti del suo template. Questa opzione può essere a volte confusionaria ed è generalmente usata solo se desideri installare ed eseguire un’applicazione due volte contemporaneamente.
Automatically open the sidebar=Apri automaticamente la barra laterale
Allows CA to automatically open the sidebar for more information when hovering over an application’s icon=Consenti a Community Applications di aprire automaticamente la barra laterale per maggiori informazioni quando si passa il mouse sull’icona di un’applicazione.
Allow CA to check for updates to applications=Controllo degli aggiornamenti delle app
This setting enables CA to check for updates to applications Note that Community Applications will always check for updates to itself=Questa impostazione consente a Community Applications di controllare gli aggiornamenti delle applicazioni. Nota: Community Applications controllerà sempre gli aggiornamenti del suo stesso plugin.
Disable Mouse Wheel Page Changes=Disabilita il Cambio Pagina con la Rotella del Mouse
This setting disables using left or right on a mouse wheel to change the page=Questa impostazione disabilita l’uso della Rotella del Mouse (a sinistra o a destra) per cambiare pagina.
Mouse wheel threshold=Soglia della Rotella del Mouse
If you tilt the mouse wheel left or right, the browser returned value will change Set the threshold to the value returned Note that different browsers and operating systems may return different values=Se inclini la Rotella del Mouse a sinistra o a destra, il valore restituito dal browser cambierà. Imposta la soglia sul valore restituito. Nota: browser e sistemi operativi diversi potrebbero restituire valori diversi.
This is a global value for all devices If it causes issues, then disable Mouse Wheel Page Changes=Questo è un valore globale per tutti i dispositivi. Se riscontri problemi, disabilita questa impostazione.
Allow CA to send any emergency notifications=Abilita Notifiche di Emergenza
This setting allows CA to periodically scan your system and alert you to the presence of=Questa impostazione consente a Community Applications di scansionare periodicamente il tuo sistema e di avvisarti della presenza di:
1 Known malware and security vulnerabilities within applications=• Malware noti e vulnerabilità di sicurezza all’interno delle applicazioni
2 Critical security vulnerabilities within Unraid OS=• Vulnerabilità di sicurezza critiche all’interno di Unraid OS
3 Any installed applications which may seriously hinder the operation of your server=• Qualsiasi applicazione installata che potrebbe impedire seriamente il funzionamento del tuo server
Save CA debugging information=Salva le informazioni di debug di Community Applications
Only enable this when advised to from the author of Community Applications=Abilita questa opzione solo quando richiesto dallo sviluppatore di Community Applications.
Enable developer mode=Abilita Modalità Sviluppatore
This unlocks some extra features within CA generally only used by developers Not recommended to enable=Questo sblocca alcune funzionalità extra all’interno di Community Applications generalmente usate solo dagli sviluppatori. Non è consigliato abilitarlo.

Disable Featured Applications=Disabilita app "In Evidenza"
This setting will disable Featured applications appearing on the home screen=Questa impostazione disabiliterà la visualizzazione delle applicazioni 'In Evidenza' nella schermata principale.
